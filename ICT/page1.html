Five Generations of Computers

  1. First Gen
  2. Second Gen
  3. Third Gen
  4. Fourth Gen
  5. Fifth Gen

First Gen (1940 - 1956) Features

  1. Vacuum tube technology
  2. Supported machine language only
  3. Very costly,expensive
  4. Generates a lot of heat
  5. It has huge size
  6. It needs AC
  7. Not portable
  8. Consumed a lot of electricity

Second Gen (1956 - 1963) Features

  1. It uses transistors
  2. It is faster than first gen computers
  3. Its is very costly
  4. It generates less heat
  5. It consumes less electricity
  6. It has smaller size

Third Gen (1964 - 1971) Features

  1. It used integrated circuit (IC)
  2. It is very fast
  3. It is costly
  4. It generates less heat
  5. Its smaller is size
  6. Lesser maintenance
  7. It support high-level language
  8. AC is required
  9. It is more reliable

Fourth Gen (1972 - 1980) Features

  1. It is very cheap
  2. It uses PCs
  3. No AC is required
  4. Very small in size
  5. It is reliable
  6. It is portable

Fifth Gen (1980 --) Features

  1. Development of true artificial intelligence
  2. Development of Natural language processing
  3. Advancement in parallel processing
  4. Advancement in super conductor unit
  5. It uses ultra large scale integration(ULSI) technology
